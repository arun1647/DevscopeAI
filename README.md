# DevScope AI 🚀

DevScope AI is a cutting-edge, interactive platform designed to visualize, analyze, and document your GitHub projects. Powered by an advanced AI (ZARA ❤️) and a stunning 3D isometric interface, DevScope AI helps you understand any repository from top to bottom.

## ✨ Features

- **3D Architecture View:** Automatically maps out the imported repository's components, showing interactive visual representations of your frontend, backend, database, and integrations.
- **ZARA ❤️ AI Chatbot:** Your personal AI assistant for project analysis. ZARA can analyze code quality, give project scores, and find potential weaknesses in real-time using the **Groq API**.
- **Auto Documentation Generation:** Instantly generate complete project documentation in Markdown formats (README, API, Database, Architecture, Deployment) and download them all in a single ZIP file.
- **Project Analytics:** Get a live project health score, complete with detailed breakdowns of weak areas and actionable suggestions for improvement.
- **Sleek UI/UX:** A stunning Glassmorphism dark-mode UI with smooth micro-animations.

## 🛠️ Technology Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Libraries:** FontAwesome (Icons), JSZip (File bundling)
- **AI Integration:** [Groq API](https://groq.com/) (Llama-3.3-70b-versatile model)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/arun1647/DevscopeAI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd DevscopeAI
   ```
3. Open `index.html` in your web browser.
4. Log in and paste a GitHub URL to start the 3D analysis!

## 🤖 ZARA AI Setup

To use the AI features locally, ensure you replace the `YOUR_GROQ_API_KEY` in `dashboard-script.js` with your actual Groq API key:

```javascript
const groqApiKey = "YOUR_GROQ_API_KEY_HERE";
```
